# 힐매트 랜딩페이지 제작 지원비 정산 보고서

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 힐매트(HealMat) 제품 랜딩페이지 제작 |
| 의뢰 기업 | 모르포 메디테크(Morpho Meditech) |
| 제작 기간 | 약 4주 |
| 제작 범위 | 반응형 웹 랜딩페이지 기획, 디자인, 개발 |

---

## 1. 프로젝트 배경 및 목적

### 1.1 배경
모르포 메디테크는 오리가미 기술 기반의 혁신적인 욕창예방 매트리스 '힐매트'를 개발한 스타트업입니다. 제품의 핵심 가치와 기술적 우수성을 효과적으로 전달하고, 잠재 고객(가정 간병인, 의료기관)과의 접점을 마련하기 위해 전문적인 랜딩페이지 제작이 필요했습니다.

### 1.2 목적
- 제품의 핵심 가치(70도 자동 자세 변환) 명확한 전달
- 브랜드 신뢰도 및 전문성 구축
- 상담 문의 전환율 향상
- 투자자/파트너사 대상 제품 소개 자료 역할

## 3. 구현 기능 상세

### 3.1 반응형 웹 디자인

#### 구현 내용
- **3단계 반응형 브레이크포인트** 적용
  - 모바일: 768px 이하
  - 태블릿: 769px ~ 1024px
  - 데스크톱: 1025px 이상
- **useMediaQuery 커스텀 훅** 개발로 실시간 뷰포트 감지
- **useResponsive 훅** 통한 isMobile, isTablet, isDesktop 상태 관리

#### 적용 영역
- 네비게이션 바: 데스크톱 메뉴 / 모바일 햄버거 메뉴 자동 전환
- 그리드 레이아웃: 3열 → 2열 → 1열 자동 조정
- 폰트 크기: 뷰포트에 따른 동적 조정
- 패딩/마진: 화면 크기별 최적화
- 이미지 컨테이너: 비율 유지 반응형

### 3.2 애니메이션 효과

#### CSS 키프레임 애니메이션 (10종)

| 애니메이션 | 설명 | 적용 영역 |
|-----------|------|----------|
| fadeInUp | 아래에서 위로 페이드인 | 버튼, 섹션 요소 |
| fadeInDown | 위에서 아래로 페이드인 | 헤더 요소 |
| fadeInLeft | 왼쪽에서 오른쪽으로 | 텍스트 블록 |
| fadeInRight | 오른쪽에서 왼쪽으로 | 이미지 블록 |
| scaleIn | 확대되며 등장 | 모달, 팝업 |
| bounceIn | 바운스 효과 등장 | 강조 요소 |
| float | 떠다니는 효과 | 테스트 결과 카드 |
| pulse | 맥박 효과 | Hero 애니메이션 |
| shimmer | 반짝이는 효과 | 로딩 상태 |
| scrollBounce | 스크롤 유도 | 스크롤 인디케이터 |

#### 제품 동작 시뮬레이션 애니메이션
- **airCellInflate**: 에어셀 팽창 시각화
- **boardTilt**: 상판 기울기 연동
- **patientTilt**: 환자 체위 변환 표현

### 3.3 호버링(Hover) 효과

#### 버튼 호버 효과 (4종)

| 클래스명 | 효과 | 적용 대상 |
|---------|------|----------|
| btn-primary | 상승 + 그림자 확장 + 빛 스와이프 | 주요 CTA 버튼 |
| btn-secondary | 상승 + 오렌지 그림자 | 서브 CTA 버튼 |
| btn-outline | 배경색 전환 + 물결 효과 | 외곽선 버튼 |
| btn-circle | 확대 효과 | 캐러셀 화살표 |

#### 카드 호버 효과 (3종)

| 클래스명 | 효과 | 적용 대상 |
|---------|------|----------|
| card-hover | 상승 + 그림자 강화 | 통계/기능 카드 |
| card-hover-lift | 상승 + 확대 | 주요 기능 카드 |
| card-glow | 테두리 글로우 | 테스트 결과 카드 |

#### 기타 호버 효과

| 클래스명 | 효과 | 적용 대상 |
|---------|------|----------|
| nav-link | 밑줄 애니메이션 | 네비게이션 링크 |
| footer-link | 이동 + 색상 변경 | 푸터 링크 |
| badge-hover | 확대 + 그림자 | 배지/태그 |
| icon-hover | 확대 + 회전 | 아이콘 |
| stat-number | 확대 + 색상 변경 | 통계 숫자 |

### 3.4 페이지 섹션 구성 (12개 섹션)

| 순서 | 섹션명 | 주요 콘텐츠 |
|-----|-------|------------|

| 6 | How It Works | 4단계 작동 프로세스 |
| 7 | Video | 제품 동작 이미지 갤러리 |
| 8 | Comparison | 경쟁 제품 비교표 |
| 9 | Business Model | B2C/B2B 전략, 인증 로드맵 |

| 13 | Footer | 연락처, 링크, 법적 고지 |

## 6. 결론 및 권고사항

### 6.1 핵심 요약
모르포 메디테크의 '힐매트' 랜딩페이지는 혁신적인 욕창예방 기술을 효과적으로 전달하기 위해 다음을 구현했습니다:

- **완전한 반응형 디자인**: 모든 디바이스에서 최적화된 사용자 경험
- **풍부한 애니메이션**: 제품의 기술적 특징을 직관적으로 시각화
- **세련된 인터랙션**: 호버 효과를 통한 사용자 참여 유도
- **전환 최적화**: CTA 배치와 Google Forms 연동으로 상담 문의 유도

### 6.2 권고사항

1. **콘텐츠 보강**: 실제 제품 사진/영상으로 플레이스홀더 교체 권장
2. **SEO 최적화**: '욕창예방 매트리스', '자동 체위변환' 등 키워드 전략 수립
3. **분석 도구**: Google Analytics 4, Hotjar 히트맵 연동으로 사용자 행동 분석
4. **A/B 테스트**: CTA 문구 및 배치 최적화 테스트 진행
5. **글로벌 확장**: 영문 페이지 제작 검토 (해외 시장 대응)
